# Entendendo Algoritmos

Repositório com meus resumos e soluções de exercícios do livro **Entendendo Algoritmos**.

## 📚 Objetivo
Documentar minha evolução em algoritmos e resolução de problemas ao longo dos capítulos do livro.

Atualmente estou no 7 período do curso de engenharia de software e sinto que o meu pensamento é api para tudo como algumas pessoas dizem, quero ir além disso e aprender como os algoritmos funcionam e como usá-los para resolver problemas do dia a dia.

## 📁 Estrutura do repositório
- `resumos/` → anotações por capítulo
- `exercicios/` → soluções dos exercícios
- `extras/` → materiais complementares

## 🚀 Como usar este repositório
1. Escolha um capítulo em `resumos/`.
2. Consulte os exercícios correspondentes em `exercicios/`.
3. Compare abordagens e tente resolver antes de ver a solução.

## 🎯 Metas de aprendizado
- Melhorar raciocínio lógico e algorítmico.
- Ganhar fluidez na análise de complexidade.
- Resolver problemas com soluções mais eficientes.
